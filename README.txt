                               CESAR.EXE


README
  LE PROGRAMME CESAR.EXE PERMET DE CHIFFRER ET DÉCHIFFRER UN TEXTE À L'AIDE
  D'UNE CLÉ. CEPENDANT IL EST AUSSI POSSIBLE D'UTILISER LA FONCTION AUTOMATIQUE
  QUI PERMETTRA DE DÉCHIFFRER AUTOMATIQUEMENT UN TEXTE QUE VOUS AVEZ SAISI, IL
  N'EST NÉANMOINS PAS FIABLE À 100%.

  VOUS TROUVEREZ CI-DESSOUS LES INFORMATIONS QUI VOUS PERMETTERONT D'UTILISER
  NOTRE PROGRAMME.

CONSEILS D'UTILISATION
  NOUS VOUS CONSEILLONS D'UTILISER NOTRE PROGRAMME SUR UN LONG TEXTE EN FRANÇAIS
  POUR QU'IL SOIT FIABLE. LE TEXTE PEUT ÊTRE RENSEIGNÉ EN MAJUSCULES COMME EN
  MINUSCULE.

OPTIONS
  -c      CHIFFREMENT D'UN TEXTE SAISI SUIVI DE LA CLÉ À RENSEIGNER

  -d      DÉCHIFFREMENT D'UN TEXTE SAISI SUIVI DE LA CLÉ À RENSEIGNER

  -a      DÉCHIFFREMENT AUTOMATIQUE D'UN TEXTE SAISI
